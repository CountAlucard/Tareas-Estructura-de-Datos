TDA Cursores

A continuacion se crea un programa en el cual podamos almacenar los titulos y los autores de diferentes libros sin
perder ninguno de ellos. El usuario sera quien suministre los datos de cada libro y asi poder almacenar la informacion.

Se utilizaran los cursores para dicho programa y sera el usuario quien decida cuantos libros desea almacenar.

Se inicializaran dos cursores:
-titulos (Para almacenar los datos de el titulo de cada libro)
-autores (Para almacenar los datos del autor de cada libro)
